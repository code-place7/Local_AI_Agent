## Features

Local LLM with Ollama ‚Äì Run AI models locally without internet dependency.
LangChain Integration ‚Äì Orchestrate prompts, memory, and tools for complex AI workflows.
Chroma Vector Database ‚Äì Store and retrieve embeddings for efficient semantic search.
Privacy First ‚Äì All data and inference remain on your device.
Customizable ‚Äì Easily switch models or add tools.

### Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/code-place7/local-ai-agent.git
cd local-ai-agent

```

###  Create a Virtual Environment
python -m venv venv
source venv/bin/activate    # Mac/Linux
venv\Scripts\activate       # Windows

###  Install Dependencies
pip install -r requirements.txt

###  Install & Run Ollama
Download Ollama from: https://ollama.ai
Run your desired model:
ollama run llama2

### Usage
python main.py
The agent will load your chosen Ollama model.
You can chat, store context, and retrieve information using Chroma.
